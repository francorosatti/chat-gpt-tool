# chat-gpt-poc

## Usage

1. Create `.env` file with the following content:
    ```dotenv
    OPEN_AI_URL="https://api.openai.com/v1"
    OPEN_AI_API_KEY=<YOUR_API_KEY>
    ```

2. Run the application:
    ```sh
   python3 src/main.py
    ```
